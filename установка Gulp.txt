node -v - проверить версию Node, чтобы понять установлен ли он или нет.
gulp -v - проверить версию Gulp, чтобы понять установлен ли он или нет.
	npm i gulp -g - если Gulp не установлен в самой системе.
npm init - Запускаем инициализацию проекта командой. Пишем основную инфу проэкта.
npm i gulp --save-dev - установим Gulp локально в папку нашего проекта. флаг —save-dev как раз говорит установщику установить gulp локально в папку нашего проекта.

npm install - если уже есть готовый файл package.json со всеми установленными пакетами (зависимости)
